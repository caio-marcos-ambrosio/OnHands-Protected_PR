# OnHands - Protected_PR
Este repositório é destinado a atividade de **DevOps**.
## Objetivo do Projeto:
Criar um site coletivo, reunindo as páginas sobre lugares dos sonhos de todos os integrantes do grupo em uma única aplicação web.

Cada integrante ficou responsável por:

Criar sua section pessoal.

Abrir a PR com suas alterações.

Seguir o fluxo correto de versionamento (pull → commit → push).

## Tecnologias Utilizadas:
    HTML5
    CSS3
    Git
    GitHub
    GitHub Pages

## Como executar o projeto localmente:

1- 
```bash
git clone https://github.com/caio-marcos-ambrosio/OnHands-Protected_PR.git
``` 
2 - Acessar a pasta do projeto.

3 - abrir o **index.html** no navegador.

## Deploy
O projeto foi publicado utilizando o **GitHub Pages**.
